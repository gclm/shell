# gclm-shell

> 常见自动化脚本集合

## 一键环境安装脚本
```
wget https://gclm.coding.net/p/shell/d/shell/git/raw/master/linux/install.sh && chmod +x install.sh && bash install.sh
```

## Docker 安装脚本
```
curl -fsSL https://get.docker.com | bash -s docker --mirror Aliyun
```
